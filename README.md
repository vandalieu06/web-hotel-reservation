## ACTIVIDAD JS POO

## 1. Utilitzant POO, realitzar una web complerta per la reserva d’habitacions d’un hotel.

### Característiques:

- Ocupació màxima de dues persones.
- Reserva per dies complerts.
- Mínim 6 habitacions diferents característiques.


### Funcionalitats:
- El client disposarà d’un calendari per reservar.
- Posarà la informació de que habitacions estan disponibles.
- Disposarà d’un formulari per registrar les dades del client.
- S’ha d’aplicar estils i fer una web agradable per l’usuari. 
- Aquesta són les funcionalitats mínimes, recordeu de comentar el codi.
